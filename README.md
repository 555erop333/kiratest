# Sunny English Garden

Красивый сайт с тестами по английскому языку для 4 класса. Внутри 9 грамматических разделов по 100 вопросов и отдельный словарный тренажёр по темам из школьной тетради: профессии, город, кемпинг, эмоции, летний отдых и другие.

## Возможности

- 9 грамматических разделов и 900 вопросов
- словарный тренажёр с выбором перевода слова по темам в обе стороны
- показ заданий по одному
- перевод предложения и вариантов ответа
- счёт `+1` и `-1`
- сохранение прогресса и лучших результатов в браузере
- адаптация под телефон и компьютер
- поддержка возврата в меню через кнопку назад

## Структура проекта

- `index.html` — разметка страницы
- `styles.css` — оформление и анимации
- `app.js` — логика тестов и сохранения прогресса
- `questionBank.js` — банк вопросов
- `vocabularyBank.js` — словарные темы и слова для повторения

## Локальный запуск

Открой папку проекта и запусти любой простой статический сервер.

Пример на Python:

```bash
python -m http.server 3000
```

После этого сайт будет доступен по адресу `http://127.0.0.1:3000`.

## Деплой на Render

Для этого проекта нужен именно `Static Site`.

Настройки в Render:

- `Build Command`: оставить пустым
- `Publish Directory`: `.`

После деплоя Render будет раздавать файлы напрямую как статический сайт, без Python и без дополнительных сервисов.
